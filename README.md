# japan-trip

Interactive Japan trip planner built for a father-son trip — Nate & Dad's Japan 2026.

## What it does

A custom travel planning web app with:
- Interactive map with pinned locations (powered by Leaflet.js)
- Day-by-day itinerary view
- Japanese-styled UI (washi paper aesthetic, Shippori Mincho + Zen Kaku Gothic fonts)
- Mobile-friendly layout

## Stack

- Pure HTML/CSS/JS — no build step, runs in any browser
- Leaflet.js for maps
- Hosted via GitHub Pages

## Why I built it

Wanted a single shareable page we could both pull up on our phones during the trip — no app installs, no logins.
